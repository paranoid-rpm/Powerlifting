# Iron Vector Lab

Вики-платформа по пауэрлифтингу с курсами под конкретные цели.

## Что внутри
- `index.html` - главная, интерактивные инструменты и быстрый вход в разделы.
- `theory.html` - теоретическая вики-база по силе и адаптации.
- `technique.html` - вики по технике приседа, жима и тяги.
- `programs.html` - вики по периодизации и построению цикла.
- `recovery.html` - вики по восстановлению и мониторингу.
- `courses.html` - авторские курсы: `Жим 100 кг`, `Присед 140 кг`, `Тяга 180 кг`.
- `media.html` - медиатека видео и фото для визуального анализа.
- `styles.css` - визуальная система и адаптивная верстка.
- `script.js` - интерактив UI (меню, фильтры, лайтбокс, анимации).

## Концепция
- Основа материалов: вики-структура с практическими выводами по видео-кейсам.
- Формат: "что делать", "что контролировать", "как корректировать".
- Каждый учебный раздел связан с видео и прикладными чек-листами.

## Локальный запуск
```bash
python -m http.server 8000
```

Открыть: `http://localhost:8000`
