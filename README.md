# Powerlifting

Многостраничный современный сайт-учебник по теме **«Пауэрлифтинг»**.

## Страницы
- `index.html` — главная и навигация по разделам.
- `theory.html` — расширенная теория силы и адаптации.
- `technique.html` — техника приседа, жима, тяги.
- `programs.html` — периодизация и шаблоны программ.
- `recovery.html` — восстановление, питание, профилактика травм.
- `media.html` — большая медиатека (видео + изображения).

## Что улучшено
- Переведено с одностраничной структуры на многостраничную.
- Добавлено больше контента и видеоматериалов.
- Усилен визуальный стиль (яркие акценты, градиенты, улучшенная компоновка).
- Сохранена адаптивность для мобильных и десктопных экранов.

## Запуск локально
```bash
python3 -m http.server 8000
```

Откройте в браузере: `http://localhost:8000`
