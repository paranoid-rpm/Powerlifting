# Железный конспект

Большой авторский сайт по пауэрлифтингу: теория, техника, программы, восстановление, курсы под цели и медиатека с видео и фото.

## Что внутри
- `index.html` - главная страница с маршрутом по сайту, калькулятором 1ПМ и генератором мини-плана.
- `theory.html` - подробный разбор логики силового прогресса, специфичности, объёма, интенсивности и пика.
- `technique.html` - техника приседа, жима и тяги: фазы, ошибки, подсказки, видеоанализ и чек-листы.
- `programs.html` - программирование цикла: накопление, интенсификация, пик, микроциклы и авто-регуляция.
- `recovery.html` - восстановление: сон, питание, стресс, разгрузка и маркеры усталости.
- `courses.html` - готовые маршруты под цели `Жим 100 кг`, `Присед 140 кг` и `Тяга 180 кг`.
- `media.html` - фильтруемая медиатека с видео и фотогалереей референсов.
- `styles.css` - общая визуальная система, адаптив и оформление всех страниц.
- `script.js` - интерактив: меню, фильтры, табы, лайтбокс, анимации, счётчики и динамический hero-текст.

## Особенности
- Более агрессивный авторский визуальный стиль вместо шаблонной академической подачи.
- Много тематического текста и связный маршрут по материалу.
- Адаптивная многостраничная структура.
- Встроенные интерактивные блоки для быстрого расчёта и навигации по теме.

## Локальный запуск
```bash
python -m http.server 8000
```

Открыть: `http://localhost:8000`
