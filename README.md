# Powerlifting

Современный одностраничный сайт-учебник по теме **«Пауэрлифтинг»**.

## Что внутри
- Большой теоретический блок по основам пауэрлифтинга.
- Разбор техники: присед, жим лёжа, становая тяга.
- Разделы по периодизации, питанию и восстановлению.
- Встроенные обучающие видео и тематические изображения.
- Адаптивный современный интерфейс.

## Запуск локально
Откройте `index.html` в браузере или поднимите локальный сервер:

```bash
python3 -m http.server 8000
```

После запуска откройте: `http://localhost:8000`
