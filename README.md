# Powerlifting

Интерактивный многостраничный сайт-учебник по теме **«Пауэрлифтинг»**.

## Страницы
- `index.html` — главная + интерактивные инструменты (1ПМ калькулятор, генератор мини-плана).
- `theory.html` — расширенная теория силы и адаптации.
- `technique.html` — техника движения + интерактивный чек-лист.
- `programs.html` — периодизация + табы по фазам цикла.
- `recovery.html` — восстановление и FAQ.
- `media.html` — видеотека с фильтрами + изображения с лайтбоксом.

## Интерактив и анимации
- Reveal-анимации блоков при скролле.
- Анимированный прогресс чтения страницы.
- Анимированные счётчики на главной.
- Кнопка «наверх».
- Параллакс-движение фоновых орб.

## Запуск локально
```bash
python3 -m http.server 8000
```

Откройте в браузере: `http://localhost:8000`
